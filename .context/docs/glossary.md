# Glossary

## LiveKit
SFU platform used for signaling and media routing.

## SFU
Selective Forwarding Unit. Routes media streams between participants without full mesh.

## Token Server
Backend service that issues LiveKit access tokens for clients.

## Guest Join
Join flow without authentication, using only a participant name.

## Lobby
Pre-join UI where users enter their name and (optionally) room name.

## Room
LiveKit room where participants connect and publish media.

## Shared Types
`packages/shared-types` module that defines Zod schemas and inferred types.

## PRD
Product Requirements Document stored at `.opencode/PRD.md`.
