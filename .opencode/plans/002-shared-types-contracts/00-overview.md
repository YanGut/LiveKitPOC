# Plano 002 - Shared Types

## Objetivo
- Centralizar contratos com Zod e tipos inferidos.
- Garantir a Regra de Ouro do Monorepo.

## Entregaveis
- `packages/shared-types` com schemas, tipos e exports.
- Build de TypeScript gerando `dist`.

## Criterios de sucesso
- Backend e Frontend importam os mesmos schemas.
- Validacao e tipos consistentes em ambos os lados.
