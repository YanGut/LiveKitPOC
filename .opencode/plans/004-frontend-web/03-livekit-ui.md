# Sala e LiveKit UI

## Conexao
- Usar `livekit-client` para conexao WebRTC.
- `LiveKitRoom` e `VideoConference` de `@livekit/components-react`.

## Estilos
- Importar `@livekit/components-styles` como CSS base.
- Complementar com `tailwindcss` se necessario.

## Estado e feedback
- Exibir estado de conexao (connecting/connected/reconnecting).
- Indicar permissao de camera/microfone quando negada.
